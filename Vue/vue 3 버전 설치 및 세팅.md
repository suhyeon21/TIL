## vue 3 버전 설치 및 세팅  
<br/>

**1. 먼저 nodeJS를 설치한다**  
<br/>

**2. vue cli 설치**
```
npm install -g @vue/cli
```
<br/>

**3. 프로젝트 생성**
```
vue create 폴더명
```
<br/>

**4. router 설치**
```
vue add router
```
<br/>

**5. 유용한 확장 프로그램**
 
  -vetur

  -html css support

  -vue 3 snippets
